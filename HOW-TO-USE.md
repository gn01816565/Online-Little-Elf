# 🎮 線上多人小精靈遊戲 - 安裝說明

## ⚠️ 重要提醒

這個遊戲**無法在 Claude.ai 的 artifact 環境中直接運行**，因為：
- Claude artifact 有嚴格的內容安全政策 (CSP)
- 無法載入 Firebase SDK
- 無法使用彈出式對話框 (alert)

## 🚀 正確的使用方式

### 方法 1：本地運行（最簡單）

1. **下載檔案**
   - 下載 `online-multiplayer-pacman-10players-fixed.html` 到你的電腦

2. **用瀏覽器直接打開**
   - 在電腦上找到檔案
   - 雙擊用 Chrome、Firefox、Edge 等瀏覽器打開
   - 這時 Firebase SDK 可以正常載入

3. **配置 Firebase**（必須）
   - 前往 https://console.firebase.google.com
   - 建立新專案
   - 啟用 Realtime Database
   - 複製配置資訊到檔案中（第 322-329 行）

4. **開始使用**
   - 一個人建立房間
   - 其他人用房間代碼加入
   - 開始遊戲！

### 方法 2：部署到網站（推薦給攤位使用）

這樣任何人都可以透過網址訪問：

#### 使用 Firebase Hosting（免費）

```bash
# 1. 安裝 Firebase CLI
npm install -g firebase-tools

# 2. 登入
firebase login

# 3. 初始化
firebase init hosting

# 4. 部署
firebase deploy --only hosting
```

完成後會得到一個網址，例如：`https://your-project.web.app`

#### 使用 Netlify（更簡單）

1. 前往 https://www.netlify.com
2. 拖曳 HTML 檔案到網站
3. 立即獲得一個網址

#### 使用 GitHub Pages

1. 上傳檔案到 GitHub repository
2. 在 Settings → Pages 啟用
3. 獲得 `https://你的用戶名.github.io/專案名` 網址

## 📱 推薦配置（給攤位使用）

### 硬體需求
- **主機設備**：1台電腦或平板（用來顯示遊戲）
- **玩家設備**：玩家用自己的手機加入

### 設定步驟

1. **準備顯示螢幕**
   - 用電腦/平板接大螢幕或投影機
   - 顯示遊戲畫面

2. **建立遊戲房間**
   - 在主螢幕上建立房間
   - 將 6 位數代碼放大顯示

3. **玩家加入**
   - 玩家用手機打開遊戲網址
   - 輸入代碼加入房間
   - 最多 10 人同時玩

4. **開始遊戲**
   - 玩家都加入後
   - 點「開始遊戲」
   - 玩家在手機上用虛擬按鍵或方向鍵控制

## 💡 故障排除

### Q: 點按鈕沒反應
A: 確認檔案是在本地打開或部署到真實伺服器，不是在 Claude artifact 中

### Q: Firebase 錯誤
A: 
- 檢查配置是否正確貼上
- 確認 Realtime Database 已啟用
- 檢查安全規則是否允許讀寫

### Q: 無法加入房間
A:
- 確認房間代碼正確（6 位數）
- 確認房主沒有離開（房主離開會關閉房間）
- 確認房間未滿（最多 10 人）

### Q: 遊戲很延遲
A:
- 選擇離你較近的 Firebase 區域
- 檢查網路連線
- 減少同時線上玩家數量

## 📞 需要協助？

2. 確認 Firebase 配置正確
3. 確認檔案在正確的環境中運行
